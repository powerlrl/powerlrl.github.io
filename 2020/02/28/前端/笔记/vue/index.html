    vue组件

    创建全局组件的两种方式

    • 利用构造器（extend）
    • vue.component方式

    局部组件的创建方式，具有独立的作用域，只能在自己内部访问，如果需要则传值解决

    • 组件的data必须是一个函数，且必须返回一个对象
    • 只能有一个根元素

    父子组件(这样就可以只暴露父组件，而不要把所有的组件都暴露出来)

    组件中可以使用template标签，来在外面定义所需要的没有高亮。

    上面的不管是什么方式都写的感觉很恶心，所以最好的解决方法是，把一各个组件都拆分成对应的.vue
