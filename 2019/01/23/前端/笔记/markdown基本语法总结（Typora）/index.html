对文字的特殊标注

标题

1-6级标题（#-######）注意#号后面有空格

下划线

可以使用超文本标记语言中的标签或者（ctrl“+u即可产生下划线

强调使用前后各2个星号

斜体

前后各一个星号

删除

sdfkjs

前后加两个波浪线（必须时英文输入法下哟）

方框

//同上时英文输入法下一旁点的点号

框起来

注释

今天天气很好啊 ^s

注意我是中括号哟！然后加上^和后面我的注释名字，然后再下面重新一下其后跟上冒号，即可注释后面的文字了。

高亮

高亮内容部分前后加入两等于号即可

==我是高亮部分==

表格

ctrl+T即可快速打开创建表格的对话框。

水果 肉类 饭
橙子 猪肉 米饭
西瓜 牛肉 稀饭
柚子 狗肉 馒头
我是ctrl+enter产生出来的行

分割线

输入三个以上的星号或者输入四个以上的减号我就会变成分割线啦

插入

图片

把图片拖进来就可以啦

链接

前面写你的链接名称（）小括号里面写你的链接地址

百度一下(http://www.baidu.com)

数学公式

代码

javascript

引用

我又打了一个大于号，嘻嘻！（以此类推）引用的嵌套。

列表

有序列表

格式：1.空格即可

  1. 神奇宝贝
  2. 神奇动物
  3. 我是一个小仙女
  4. 我可是要喝露水长大的

无序列表

格式：+/-/*其后是空格都是无序列表的开头

  • 好好学习
  • 身体健康
